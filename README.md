```bash

```

## Preferences
| Preference       | Default                  |
| ------------- | ---------------------- |
| **`ONYX_ANDROID_SDK_VERSION`** | <code>`7.1.1`</code> |
| **`ONYX_IOS_SDK_VERSION`**   | <code>`7.1.10`</code>   |
